# Outlast CTL

Command-line tool for interacting with the Outlast API.

## Installation

```bash
npm install -g @outlast/ctl
```

## Configuration

Set the following environment variables:

```bash
export OUTLAST_API_URL=http://localhost:3000
export OUTLAST_CREDENTIALS=admin:secret
```

Or create a `.env` file in the project root.

## Commands

### records:create

Create a new record interactively.

```bash
ol records:create
```
